# Dotfiles

#### A curated collection of dotfiles used for software development.

Basic usage example:

``` sh
cd ~/ && git clone DOTFILES.git dotfiles

mkdir -p .config && cd .config && ln -s ../dotfiles/.config/nvim nvim
```

### Entry points

#####  Neovim: `./config/nvim/init.vim`
#####  ZSH: `/config/zsh/.zshrc`
